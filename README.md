# 欢迎使用iViewer

## 简介

- iViewer是一款免费、跨平台、轻量高效的专业医疗影像浏览器；
- 支持符合DICOM 3.0标准的医学影像，包括CT、MR、DX、US、DR等；
- 提供丰富的图像浏览功能，包括布局、平移、缩放、旋转、反色、播放、窗宽窗位调整、测量、同步等。

## 环境要求

- 支持macOS11及以上，推荐macOS13及以上
- 支持windows7及以上，推荐windows10及以上
- 支持Ubuntu 20.04 LTS及以上

## 声明

- 本软件可离线使用，不收集任何用户DICOM数据

# 图像工具

iViewer图像工具说明，具体如下：

| 工具         | 功能描述                                                                                                                               |
|--------------|----------------------------------------------------------------------------------------------------------------------------------------|
| 重置         | 单击后，图像窗口内所有图像状态(窗宽/窗位、平移、缩放、旋转)恢复为默认，测量结果不清除                                                   |
| 布局         | 调整序列窗口布局，设置常见1、2、4、6、9个窗口布局，支持自定义窗口布局                                                                   |
| 移动         | 激活状态下，左键按下拖动平移图像窗口内图像                                                                                             |
| 缩放         | 激活状态下，左键按下拖动放大缩小图像窗口内图像                                                                                         |
| 窗宽窗位     | 激活状态下，左键按下拖动调整窗宽/窗位<br>下拉菜单内提供预设的窗宽/窗位：<br>肺（W1400 L-500） ；纵隔（ W350 L40）；头（W80 L40）；骨（W1500 L300）；且支持自定义预设窗宽/窗位 |
| 反色         | 激活序列窗口内所有图像反色显示                                                                                                         |
| 旋转         | 激活序列窗口内图像左转90°<br>下拉菜单内工具包含左转90°；右转90°；水平翻转；垂直翻转                                                              |
| 测量         | 单击激活直线测量<br>下拉菜单内工具包含：<br>- 长度测量（激活直线测量，可在图像上拖拽生成直线，并显示测量结果）<br>- 角度测量（激活角度测量，可在图像上拖拽两条直线，并显示测量结果）<br>- 像素测量（激活像素测量，可在图像上单击生成测量点，并显示测量结果）<br>- 椭圆测量（激活椭圆测量，可在图像上拖拽生成椭圆，并显示测量结果）<br>- 矩形测量（激活矩形测量，可在图像上拖拽生成矩形，并显示测量结果）<br>- 箭头标注（激活箭头标注，可在图像上拖拽生成箭头，输入标注，生成注释）<br>- 放大镜（激活放大镜，并且可以调节放大镜倍数和尺寸）<br>- 清空测量（单击后，将清空此图像所有测量数据） |
| Dicom信息    | 显示当前激活窗口图像的DICOM信息<br>下拉菜单内工具包含：<br>- DICOM信息列表（单击后，弹出DICOM Tags信息弹窗，展示DICOM信息）<br>- 信息隐藏（显示或隐藏图像区域四角信息） |
| 同步         | 在多窗格状态下，位置/索引同步激活后，显示序列自动跳转到与焦点框内图像所在层面位置/索引一致的图像，并保持同步显示                         |
| 排序         | 序列排序功能支持以下两种方式：<br>- Instance Number排序<br>- Slice Location排序                                                              |
